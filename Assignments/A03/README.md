# A03 – Case Study Analysis: Edge-Computing Video Analytics

## Objective
The goal of this assignment was to analyze a real-world edge computing system used for traffic monitoring in smart cities.

## Case Study Summary
This case study focuses on the Liverpool Smart Pedestrians project, which uses edge computing and video analytics to monitor pedestrian and vehicle traffic in real time.

The system processes video data locally using AI models, ensuring privacy by avoiding transmission of raw video footage.

## Key Technologies
- NVIDIA Jetson TX2 (edge device)
- YOLOv3 (object detection)
- SORT algorithm (object tracking)

## My Analysis
The project demonstrates how edge computing can be applied in real-world smart city environments.

Key strengths:
- Real-time processing
- Privacy-focused design
- Scalable architecture

Key challenges:
- Accuracy in dense environments
- Hardware limitations
- CPU bottlenecks

## What I Learned
This assignment helped me understand how AI systems are deployed outside of controlled environments.

I learned how edge computing reduces latency, improves privacy, and enables real-time decision making in smart cities.
