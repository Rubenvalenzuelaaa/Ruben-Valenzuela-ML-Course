# L02 – Deploying an AI Model on a Simulated Edge Device

## Objective
The purpose of this lab was to train a machine learning model and deploy it using TensorFlow Lite in a simulated edge environment.

## What I Did
- Trained a neural network using the MNIST dataset
- Evaluated model accuracy
- Converted the model to TensorFlow Lite format (.tflite)
- Deployed the model using the TFLite Interpreter
- Performed inference on test data

## Key Learning
This lab helped me understand the full lifecycle of an AI model:
training → optimization → deployment.

I learned that model deployment is just as important as training,
especially for real-world applications on mobile and IoT devices.
