# L03 – Deploying a Simple AI Model on a Simulated Edge Device

## Objective
The goal of this lab was to train a Convolutional Neural Network (CNN) and deploy it in a simulated edge computing environment using TensorFlow Lite.

## What I Did
- Loaded and preprocessed the MNIST dataset
- Built and trained a CNN model
- Evaluated model accuracy
- Converted the model to TensorFlow Lite (.tflite)
- Ran inference using the TFLite interpreter
- Measured inference latency

## Key Results
- Training Accuracy: ~99.6%
- Validation Accuracy: ~98.6%
- Average Inference Latency: ~0.38 ms

## What I Learned
This lab helped me understand how machine learning models can be deployed on edge devices.

I learned how TensorFlow Lite reduces model size and improves performance, making it suitable for real-time applications like IoT systems and image recognition.
