# A01 – Understanding Development Environments and Tools for Edge AI and IoT

## Objective
The purpose of this assignment was to explore important development tools used in Edge AI and IoT projects.

## Summary
This report discusses tools such as Visual Studio Code, Node.js, Edge Impulse CLI, TensorFlow/TensorFlow Lite, Google Colab, and generative AI coding assistants. Each tool supports a different part of the Edge AI workflow, from writing code and collecting data to training models and deploying them on small devices.

## Tools Covered
- Visual Studio Code
- Node.js
- Edge Impulse CLI
- TensorFlow and TensorFlow Lite
- Google Colab
- GitHub Copilot and other AI coding tools

## What I Learned
I learned that Edge AI development requires more than just machine learning knowledge. It also requires understanding hardware limits, lightweight model deployment, data collection, and the tools needed to move from experimentation to real-world device deployment.

## Challenges
One challenge was understanding how all the tools connect together in one workflow. For example, a model may be trained in Google Colab, converted with TensorFlow Lite, and then deployed to an edge device using another tool.
