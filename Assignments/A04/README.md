# A04 – Reflective Journal on IIoT Protocols Project

## Objective
The purpose of this assignment was to reflect on the design and simulation of an Industrial Internet of Things (IIoT) sensor network using MQTT, CoAP, and OPC UA.

## What I Did
- Supported the data pipeline and visualization layer
- Worked with MQTT broker data extraction
- Implemented real-time visualization using Matplotlib
- Validated JSON data formatting
- Tested protocol data flow and subscriptions

## What I Learned
This project helped me understand how industrial communication protocols operate in real environments.

I learned:
- MQTT uses a publish-subscribe model with a broker
- CoAP is optimized for low-power devices using UDP
- OPC UA provides secure and structured industrial communication

## Challenges
- JSON parsing errors during real-time data handling
- Performance issues with continuous data updates

## Solutions
- Improved payload validation
- Implemented rolling data window for better performance

## Future Applications
This knowledge can be applied to:
- Smart factories
- Industrial monitoring systems
- IoT dashboards
- Predictive maintenance systems
